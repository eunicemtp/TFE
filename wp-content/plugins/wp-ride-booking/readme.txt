== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Find the Plugin "WP Ride Booking" in Plugins and Activate it.
3. Use the Settings->WP Ride Booking to configure the booking settings.
